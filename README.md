# Boilerworks Go + HTMX

> Lightweight Go backend with HTMX and Templ for performance-critical server-rendered apps.

**Status:** Planned

Go with Chi or Echo router, Templ for type-safe HTML templates, and HTMX for dynamic updates. For teams that want Go's performance and simplicity with minimal frontend overhead. Ideal for high-throughput internal tools, monitoring dashboards, and lightweight services with a UI.
